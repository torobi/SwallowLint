# SwallowLint

It is a simple, easily extensible linter for Swift.

## USAGE

```
OVERVIEW: simple swift linter.

USAGE: swallow-lint [--config <config>] [<path>]

ARGUMENTS:
  <path>

OPTIONS:
  --config <config>       Config file path. (default: .swallowlint.yml)
  -h, --help              Show help information.
```
